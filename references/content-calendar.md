# Content Calendar

> Your dated editorial plan. Derived from `references/positioning.md` (Weekly content calendar
> themes + Focus areas + Content pillars). Filled during `/onboard` (the storytelling-strategist
> sequence) and editable anytime. `/post` reads the row for the target DATE to build the post brief;
> `/start-day` surfaces today's planned post.
>
> Each row carries the day's **Theme**, **Pillar focus**, **Objective** (why this theme), and the
> specific **Topic / Angle**, so `/post` can generate the right post for a date without re-deciding
> the angle. Coaching = Mode 1; financial topics flip to Mode 2 (concept-only).
